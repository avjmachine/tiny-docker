# Tiny Docker

A step-by-step project to build a minimal container runtime in Go.

## Goals
- Learn Go fundamentals through systems programming
- Understand Docker internals (namespaces, cgroups, chroot)
- Build comfort with Linux process isolation and resource control

- [ ] Part 1: Run a process and enter new namespaces
